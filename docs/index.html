Sub-IoT

Sub-IoT houses an implementation of the DASH 7 Alliance protocol, which can optionally be combined with a 3rd party implementation LoRaMAC-node of the LoRaWAN specification, for ultra low power, mid-to-long-range IoT sensor networks. The goal is to provide the reference implementation of the protocol, focusing on completeness, correctness and being easy to understand.

Get started!

Fully bidirectional

Supports push and pull communication. From sensor-to-cloud to ad hoc querying of the network, or a combination.

Low power

DASH7 allows for multi year battery lifetime, without giving up on low latency downlink communication.

sub-GHz bands

Using the license free sub-GHz bands allows for increased range and worldwide operation.

All OSI layers

DASH7 stack spans all OSI layers. All functionality is included in one spec, allowing for easy integration.

Multiple platforms

Sub-IoT is easily portable to other hardware platforms and comes with support for multiple platforms out of the box.

Fully open source

Sub-IoT is a fully open source DASH7 implementation, hosted on Github.
